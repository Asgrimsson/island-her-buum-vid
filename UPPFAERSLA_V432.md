# Uppfærsla í v4.3.2 — Speed Mode + Lazy Layers

## Markmið

- Vefurinn opnist hraðar á Render.
- Aðalkortið hleður létt.
- Þung kortalög eru sótt þegar notandi biður um þau.
- Minni líkur á 502 villu þegar kveikt/slökkt er á lögum.

## Sjálfgefið á Íslandskorti

Kveikt:
- Staðir og bæir
- Færð
- Vallaskóli

Slökkt þar til ýtt er á hnapp:
- Myndavélar
- Skjálftar
- Umferðartölur
- Vatnafar
- Nákvæmar vegalínur
- Veður

## Nýir hnappar

- Myndavélar
- Skjálftar
- Umferð
- Vatnafar
- Vegalínur
- Létt kort

## Uppfærsla

Afþjappaðu zip-skránni og settu skrárnar yfir núverandi verkefnamöppu. Keyrðu svo:

```bat
git add .
git commit -m "Speed mode lazy layers"
git push
```

Render deployar sjálfkrafa.
